## 简体中文

### 项目简介

ResumeToJob 是一个免费开源的在线简历制作工具，帮助求职者快速创建专业、美观的简历。支持多份简历管理，多种模板选择、实时编辑、PDF 导出等功能，同时保护您的隐私数据安全。

> ⚠️ **重要提示**：简历数据存储在浏览器本地，以防意外情况丢失，建议使用【管理简历】功能导出保存

## English

### Project Introduction

ResumeToJob is a free and open-source online resume builder that helps job seekers quickly create professional and beautiful resumes. It supports management of multiple resumes, multiple template options, real-time editing, PDF export, and always protects your privacy and data security.

> ⚠️ **Important Notice**: Resume data is stored in browser locally. To prevent accidental loss, it is recommended to use the [manage resumes] function to export and save
